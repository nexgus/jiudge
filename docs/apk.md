這隻 APK 只支援規劃路徑與查詢圖例, GPS 還沒加入, 會直接跳到八里觀音山.

提醒
1. 這支 APK 用 debug 簽章 (見 app/build.gradle.kts 的 buildTypes.release 沿用 debug signing). 能正常安裝與使用.
2. 首次啟動會在 App 內下載地圖資料 (basemap 約 298 MB 等), 需要網路與足夠儲存空間 — 這是設計如此 (APK 不內含圖資). 對方要有心理準備首次開啟會花時間下載.
3. 架構為 arm64 等通用, 一般現代 Android 手機 (minSdk 26, 即 Android 8.0 以上) 都可安裝.
4. Google Play Protect 可能對未知來源 APK 跳一次掃描提示, 選「仍要安裝」即可.

用一般的「側載 (side-load)」方式安裝即可, 不需要 adb 或電腦. 步驟:
1. 用檔案管理員找到該 APK, 點它.
2. 第一次安裝會跳出「不允許安裝未知來源應用程式」的提示 — 點進設定, 對「執行安裝的那個 App」(通常是檔案管理員或瀏覽器) 開啟「允許安裝未知應用程式」.
3. Android 8.0 以上是「以 App 為單位」授權, 不是早期的全域開關.
4. 回到安裝畫面, 點「安裝」, 完成.
